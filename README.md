# Terraform AWS Practice

## Overview
Provisioned AWS EC2 instance using Terraform IaC on AWS (ap-south-1 Mumbai region).

## Tools Used
- Terraform v1.15.7
- AWS EC2
- AWS IAM

## Commands Used
```bash
terraform init
terraform plan
terraform apply
terraform destroy
```
